# AI剧本创作-模块设计

此模块为AI微电影创作项目中的 AI剧本创作模块，核心功能涵盖了剧本创作的五个阶段：

**` 创意 -> 提纲 -> 桥段 -> 细节 -> 精修 `**，

并遵循AI剧本创作的7个工作流程：
```
1. 设置主题背景
2. 编写剧情梗概
3. 设计角色
4. 编写故事大纲
5. 编写开头章节
6. 逐章完稿
7. 审核修订 
```

## 功能流程设计
### 1. 创意阶段
创意阶段是AI剧本创作项目的第一个阶段，需要由AI辅助形成一个不错的创意，交互模式为互动式，可以直接由AI给出，然后由人判断题材创意是否可行；
也可以由人给出题材创意，AI辅助分析、完善，然后给出结果，供人判断是否可行。此过程可能会经过多轮Chat，直到敲定出一个可行的创意方案；

### 2. 编写主题背景
此阶段AI会根据上一步得到的创意，生成一个主题背景，并给出一个背景描写，由人工交互式、定点改进，或直接手动修改。

### 3. 编写剧情梗概
此阶段AI会根据上步得到的主题背景，生成一个剧情梗概，并给出一个梗概描写，由人工交互式、定点改进，或直接手动修改。

### 4. 设计角色
此阶段AI会根据上步得到的主题背景和剧情梗概，生成一个角色列表，包括角色的姓名、性格标签描写、年龄、性别、角色设定、技能、在故事中的定位、人物关系等等，
由人工交互式、定点改进，或直接手动修改。

### 5. 创建故事大纲
此阶段AI会根据上步得到的主题背景、剧情梗概和角色列表，生成一个故事大纲，大纲的常见结构如：
```text
引 -> 起 -> 承 -> 转 -> 合
```
或：
```text
开始 -> 发展 -> 高潮 -> 结局
```
每个节点设置若干个桥段，每个桥段有一个标题总结主要内容，桥段之间包含剧情发展的逻辑关系，一个桥段视为一个章节或者一幕，后续的具体剧本内容都填写在这些桥段里面。

### 6. 创建开头章节
此阶段AI会根据上步得到的主题背景、剧情梗概、角色列表，根据故事大纲编写第一个桥段里的具体剧本内容，支持由人工交互式、定点改进，或直接手动修改。

### 7. 逐章完稿
此阶段AI会根据上步得到的主题背景、剧情梗概、角色列表，根据故事大纲编写后续的桥段里的具体剧本内容，需要确保故事剧情前后衔接。支持由人工交互式、定点改进，或直接手动修改。
