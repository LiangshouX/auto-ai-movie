# AI剧本创作-模块设计

此模块为AI微电影创作项目中的 AI剧本创作模块，核心功能涵盖了剧本创作的五个阶段：

**` 创意 -> 提纲 -> 桥段 -> 细节 -> 精修 `**，

并遵循AI剧本创作的7个工作流程：
```
1. 设置主题背景
2. 编写剧情梗概
3. 设计角色
4. 编写故事大纲
5. 编写开头章节
6. 逐章完稿
7. 审核修订 
```

## 功能流程设计
### 1. 创意阶段
创意阶段是AI剧本创作项目的第一个阶段，需要由AI辅助形成一个不错的创意，交互模式为互动式，可以直接由AI给出，然后由人判断题材创意是否可行；
也可以由人给出题材创意，AI辅助分析、完善，然后给出结果，供人判断是否可行。此过程可能会经过多轮Chat，直到敲定出一个可行的创意方案；

### 2. 编写主题背景
此阶段AI会根据用户输入的创意，生成一个主题背景，并给出一个背景描述，供用户选择，用户选择后，AI会生成一个背景图片，供用户查看，用户确认后，AI会生成一个背景音乐，供用户查看，用户确认后，AI会生成一个背景视频，供用户查看，用户确认后，AI会生成一个背景场景，供用户查看，用户确认后，AI会生成一个背景
